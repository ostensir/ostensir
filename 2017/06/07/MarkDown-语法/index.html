MarkDown 语法

标题

atx 形式标题(1 到 6 个 # 号，代表 1 至 6 级标题)
# 一级标题
…
###### 六级标题

说明：
如果你想看起来更美观一点，可以在行尾添加数量不定的 # 号

# 一级标题 #
###### 六级标题 ######

区块引用

  1. 正常引用(每个行前加上 > 号)

    > 内容
    > 内容
    >
    > 内容

  2. 懒人引用(在一个段落前只用一个 > 号)

    > 内容
    内容
    >
    > 内容
    内容

  3. 嵌套引用(使用连续多个 > 号来形成多级引用)

    > 内容
    >
    > > 嵌套内容
    >
    > 内容

  4. 区块里包含其他 Markdown 语法

    > ## 这是一个标题。
    >
    > 1. 这是第一行列表项。
    > 2. 这是第二行列表项。
    >
    > 给出一些例子代码：
    >
    > hexo new filename

说明:
使用区块引用会自动左对齐文本
如果 > 后面是一个空行，空行将被保留

列表

  1. 有序列表(使用数字，紧接一个英文句点 .)

    1.内容
    2.内容
    3.内容

  2. 无序列表(使用 * 、+、- 号)

    + 内容
    + 内容

    - 内容
    - 内容

    * 内容
    * 内容

说明：
对于有序列表，数字可以随意写，但确保第一个列表为 1，输出结果会自动更改序号

例如：

1.内容
8.内容
4.内容

输出结果：

1.内容
2.内容
3.内容

列表会自动对齐，但如果你想更美观一点，除了序号行，其他行可以缩进最多三个空格

例如：

1.内容
内容
2.内容
内容

可以缩进三个空格来美观

1.内容
内容
2.内容
内容

无论以上哪种，输出结果自动对齐：

1.内容
内容
2.内容
内容

如果两个列表项的内容之间是用空行分开的，空行会被保留
列表标记后面只能是一个空格或者一个制表符

1.内容
* 内容

如果列表项内要放入引用，则 > 号也必须缩进，即一个制表符

1.内容
> 引用
> 引用

如果列表项内要放入代码块的话，代码就要被缩进两次，8 个空格或 2 个制表符

1.内容
代码

包含段落，每个列表项下的每个段落可以缩进 4 个空格或者一个制表符

1.段落1
段落1

段落2
段落2
2.内容

代码块

  1. 代码缩进一个制表符

    内容：
    代码

  2. 三个反引号，输出结果的代码附加序号

    ```
    代码1
    代码2
    ```

分割线 —- 星号、减号、底线

* * *
***
*****
- - -
-————————————–

说明：
行内不能有其他东西，但可以星号或减号中间插入空格

链接(行内式、参考式，不管哪一种，链接文字用 [ ] 括起来)

  1. 行内式(在方括号紧接着圆括号内插入网址或链接)

    [要被点击的内容] (被点击后要跳转的链接)

    要在鼠标停留时显示链接的标题则用双引号引用链接里的标题

    [要被点击的内容] (这是一个链接“这是网站的标题“ )

    如果要跳转的链接是站内地址，可以使用相对路径

    跳转[关于页面] (/about/)

  2. 参考式(在方括号内再加上方括号，然后在文件任意位置定义或指出链接)

    [这是要被点击的内容\] [这是内部标记名id]
    在文件任意位置例如文件结尾
    [内部标记名id]: 链接地址 “需要还可包括标题”

  3. 隐式链接(根据第一个方括号的内容来寻找定义的链接)

    [Google] [ ]
    定义内容：
    [Google]: https://www.google.com/

注意：
参考方括号前面可以选择性的加上最多三个空格
格式顺序

方括号后面要接着冒号
冒号后面一个空格或制表符
链接地址
选择性的添加标题内容，可用双引号，圆括号包围起来

链接地址可以用 <括起来>
可以把标题放在下一行，甚至还可缩进，用于解决网址过长的美观
定义的链接不会包含在网页上的内容中
要被点击的内容中可以有字母、数字、空白、标点符号、但是不区分大小写

强调(* 、_ 、作为标记强调字或词)

  1. 一个 * 和 一个 _

    *内容*
    _内容_

  2. 两个 * 和 两个_

    **内容**
    __内容__

注意：
强调可以插在文字中间
如果 和 _ 两边都有空白的话，就被当成普通符号 反斜杠可以解决符号被当成内部代码的问题从而输出符号*

标记代码

如果要标记代码行内一小段代码，用 反引号 `包围起来`

Use the `printf()` function.

如果要在输出结果中附带 号，则使用 `` 多个 反引号

`` 代码`abc`add``
输出结果为：代码`agc`add

图片链接，参考链接那部分

只需在图片前面附加 ! 号，形式如下

! [图片名] (图片地址)

也可以参考形式和隐式链接

![图片名][id] 或者 ![图片名][]
然后定义
[id]: 图片地址 “图片名” 或者 [图片名]: 图片地址

自动链接(将文字转换为链接)

< http://www.google.com > 带下划线

\ 反斜杠用于转义一系列无法输出的标点符号
